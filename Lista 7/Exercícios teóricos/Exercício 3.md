3- O que as linhas abaixo fazem? Escreva como um comentário.
```C
int a=5, b=12; //Declarando duas variáveis de tipo inteiro, a e b, sendo a = 5 e b = 12
int *p; //Declarando uma variável do tipo inteiro ponteiro p
int *q; //Declarando uma variável do tipo inteiro ponteiro q
p = &a; //Apontando o ponteiro p para o endereço da variável a
q = &b; //Apontando o ponteiro q para o endereço da variável b
int c = *p + *q; //Declarando a variável c equivalente ao conteudo que p aponta + o conteudo que q aponta
```
